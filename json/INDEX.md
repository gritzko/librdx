# JSON Header Index

| Header | Description |
|--------|-------------|
| BASON.h | Binary JSON format with TLKV records, stack-based traversal, and page-indexed containers |
| BIFF.h | BASON merge (right-wins) and diff (produce minimal patch) operations |
| TLKV.h | Type-Length Key-Value record encoding with short (2B) and long (6B) header forms |
